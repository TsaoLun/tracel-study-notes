# 作业文件

每章作业对应一个独立的 Cargo 项目，内含可运行的测试骨架和 `todo!()` 标记。

## 前置条件

项目根目录下需已 clone cubecl 仓库：

```bash
git clone https://github.com/tracel-ai/cubecl.git
```
